# Judge Cyber Service

Specialized evaluation engine for cybersecurity challenges using Scapy for packet analysis and network security validation.

## Responsibilities

- Network packet capture and analysis
- Security exploit validation
- Network protocol validation
- Cybersecurity challenge evaluation
- Packet manipulation and validation

## Technology

- Python FastAPI
- Scapy for packet manipulation and analysis
- Network tools and utilities
- Docker for isolated network environments

## Structure

- `src/`: Python source code
- `Dockerfile`: Container build configuration
