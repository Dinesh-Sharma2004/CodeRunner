# Frontend Services

React-based frontend applications for the Universal Contest Platform.

## Applications

- **creator-studio/**: Admin interface for creating and managing contests, problems, and platform configuration
- **contest-arena/**: Participant interface for competing in contests, submitting solutions, and viewing leaderboards

## Technology

- React with TypeScript
- Modern build tools (Vite/Webpack)
- State management (Redux/Zustand)
- UI component library

## Structure

- `apps/`: Individual frontend applications
- `Dockerfile`: Container build configuration for serving the frontend
