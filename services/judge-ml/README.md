# Judge ML Service

Machine Learning evaluation engine using Papermill for executing and validating ML notebook submissions.

## Responsibilities

- Jupyter notebook execution and validation
- ML model evaluation (accuracy, precision, recall, etc.)
- Training data validation
- Model performance metrics calculation
- ML challenge scoring

## Technology

- Python FastAPI
- Papermill for notebook execution
- Jupyter kernel management
- ML libraries (scikit-learn, pandas, numpy)

## Structure

- `src/`: Python source code
- `Dockerfile`: Container build configuration
