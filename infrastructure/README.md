# Infrastructure

This directory contains infrastructure-as-code configurations for deploying and managing the Universal Contest Platform.

## Structure

- **k8s/**: Kubernetes manifests for deploying services, namespaces, and configurations
- **elk-stack/**: Elasticsearch, Logstash, and Kibana configurations for centralized logging
- **prometheus/**: Prometheus and Grafana configurations for monitoring and alerting

## Usage

These configurations are used to deploy the platform to Kubernetes clusters and set up observability infrastructure.
